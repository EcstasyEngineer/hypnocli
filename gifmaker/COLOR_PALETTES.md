## Tips

- **2 colors**: Simple, fast oscillation
- **3-4 colors**: Balanced rhythm (recommended)
- **5+ colors**: Slower, more complex patterns
- **Frame count**: Use multiples of color count for smooth loops (e.g., 20 frames with 4 colors)
- **Contrast**: Higher contrast = more intense (be cautious with seizure risk)
- **Hue spacing**: Colors evenly spaced around the color wheel create smoother transitions
