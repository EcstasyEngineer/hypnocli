1. **User-facing descriptions** - what state to expect

Scripts leave listeners in one or more of these state categories:

| **empty** | Blank, no thoughts, void |

| **owned** | Belonging to authority, possessed |
| **desperate** | Aching, needing, craving |

## Bot Implementation Notes

For Discord bot playlist generation:

1. **State tracking**: Track cumulative ending states across playlist
2. **Transition scoring**: Use matrix to score proposed next tracks
3. **Intensity ramping**: Don't jump from Light to Extreme
4. **User preferences**: Weight toward user's selected themes
5. **Avoid repeats**: Same script shouldn't appear twice in short playlist

### Suggested Algorithm
```python
def next_track(current_state, user_themes, played_already):
    candidates = [t for t in all_tracks if t not in played_already]
    candidates = [t for t in candidates if transition_score(current_state, t) >= YELLOW]
    candidates = sorted(candidates, key=lambda t: (
        theme_match_score(t, user_themes),
        transition_score(current_state, t),
        intensity_appropriateness(current_state, t)
    ), reverse=True)
    return candidates[0]
```

## Dominant Title Placeholder System

Scripts can use `[dominant]` placeholders for flexible dominant titles (Master, Goddess, etc.).

### How It Works

1. Scripts use `[dominant]` where the dominant's title should appear
2. Build command specifies `--dominant-title Master` or `--dominant-title Goddess`
3. Build system replaces all `[dominant]` with specified title before TTS rendering

### Quality Checks

The build system (`scratch/session/build_session.py`) automatically:
- **FAILS** if `[dominant]` found in any script without `--dominant-title` specified
- **WARNS** if gendered pronouns (he/him/his/she/her) found near `[dominant]`

### Scripts Using Placeholders

| Script | Placeholder Usage |
|--------|-------------------|
| harem | `[dominant]'s presence`, `[dominant]'s awareness` |
| wakener | `I am [dominant]'s good girl` trigger |

### Example Build Commands

```bash
# Master variant
python scratch/session/build_session.py my_session intro harem wakener --dominant-title Master

# Goddess variant
python scratch/session/build_session.py my_session intro harem wakener --dominant-title Goddess
```
